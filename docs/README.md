# HF-openVR-core

## Что это?
Самодельный VR шлем на Orange Pi 5 который создаеться мною и моими друзьями!
- Hegir-FT (Я)
- y44sha (MemeMaster)
- Spikelol52 (Spike)
- Мы попытаемся каждый день обновлять этот проект!

# Компоненты для VR шлема

## Основные компоненты

### 1. Вычислительная платформа
**Orange Pi 5 (4GB RAM)**
- **Цена:** 5000₽
- **Где купить:** Avito
- **Примечание:** Нужна версия с 8GB RAM, 4GB недостаточно для VR только для теста (как у меня)

### 2. Дисплеи
**2x LCD дисплей 5.5" 1920x1080**
- **Цена:** 3200₽ каждый (6400₽ за два)
- **Где купить:** Avito (с рук, от сломанных телефонов)
- **Модель:** LG LP156WF1-SPB1
- **Характеристики:**
  - Разрешение: 1920x1080
  - Размер: 5.5 дюймов
  - Интерфейс: HDMI через контроллер
  - Тип матрицы: IPS
- **Примечание:** Нужно покупать 2 одинаковых дисплея

### 3. Контроллеры дисплеев
**2x HDMI to LCD Controller Board**
- **Цена:** 1500₽ за два
- **Где купить:** AliExpress
- **Модель:** для конкретной матрицы LG LP156WF1
- **Примечание:** Контроллер должен подходить к вашей модели дисплея

### 4. Линзы
**Френелевские линзы 45мм**
- **Цена:** 500₽ пара
- **Где купить:** AliExpress
- **Характеристики:**
  - Диаметр: 45мм
  - Фокусное расстояние: 40мм
  - Тип: Френелевские
- **Примечание:** Лучше брать 2 пары на случай повреждения

### 5. Питание
**Power Bank 20000mAh с PD 12V**
- **Цена:** 3000₽
- **Где купить:** DNS/М.Видео
- **Модель:** Xiaomi Power Bank 3 Pro
- **Характеристики:**
  - Емкость: 20000mAh
  - Выход: 12V/3A через Power Delivery
  - Вес: 350г

### 6. Кабели
- **2x HDMI кабель 0.5м** - 600₽
- **USB-C кабель для питания** - 300₽
- **USB-C → USB-A для данных** - 200₽

### 7. 3D печать
**PLA филамент 1.75мм 1кг**
- **Цена:** 1500₽
- **Где купить:** Ozon/Wildberries
- **Цвет:** черный или серый
